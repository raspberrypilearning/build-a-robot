## وضع فمٍ للروبوت

لنضع فمًا للروبوت!



+ أضف تعليمة CSS البرمجية هذه لتحديد نمط الصورة `mouth1`:

```
## mouth1 {
    width: 50px;
    position: absolute;
    top: 200px;
    left: 200px;
}
```

+ يبدو فم الروبوت صغيرًا إلى حد ما، كما أنه ليس في المكان الصحيح.
