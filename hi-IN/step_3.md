## अपने रोबोट का मुंह बनाना

चलिए आपके रोबोट का मुंह बनाएँ!



+ अपनी `mouth1` छवि को स्टाइल देने के लिए, निम्नलिखित CSS कोड जोड़ें:

```
## mouth1 {
    width: 50px;
    position: absolute;
    top: 200px;
    left: 200px;
}
```

+ आपके रोबोट का मुंह काफी छोटा दिख रहा है, और सही जगह पर नहीं है।
