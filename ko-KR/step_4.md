## 로봇의 입 만들기

로봇에게 입을 만들어 줍시다!

- 이 CSS 코드를 `style.css`의 가장 아래쪽에 추가해 주십시오. 이 코드는 `mouth1` 이미지를 스타일링합니다.
    
        #mouth1 {
            width: 50px;
            position: absolute;
            top: 200px;
            left: 200px;
        }
        

그런데 로봇의 입이 너무 작은데다, 위치까지 부적절하네요!
