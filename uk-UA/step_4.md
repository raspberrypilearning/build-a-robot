## Дайте Вашому роботу рот

Давайте дамо Вашому роботу рот!

- Додайте наступний код CSS у нижній частині ` style.css </ 0>, щоб створити своє зображення  <code>mouth1`:
    
        #mouth1 {
            width: 50px;
            position: absolute;
            top: 200px;
            left: 200px;
        }
