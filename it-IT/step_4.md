## Disegnare la bocca del robot

Diamo la bocca al tuo robot!

- Aggiungi il seguente codice CSS nella parte inferiore di `style.css` per modellare l'immagine della `mouth1`:
    
        #mouth1 {
            width: 50px;
            position: absolute;
            top: 200px;
            left: 200px;
        }
